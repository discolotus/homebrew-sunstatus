# SunStatus Homebrew Tap

Install SunStatus with Homebrew:

```sh
brew install --cask discolotus/sunstatus/sunstatus
```

Upgrade after new releases:

```sh
brew update
brew upgrade --cask sunstatus
```

Uninstall:

```sh
brew uninstall --cask sunstatus
```

SunStatus source and releases live at https://github.com/discolotus/SunStatus.
